NOTE PARTNER SYN ACK
TEST MRD

NOTE JUMPIF WRONG_NUMBER
FJMP REDIALER

NOTE ACK PARTNER CHANNEL
VOID M

NOTE SEEK EOF
SEEK 9999

MARK WRITENUMBER
@REP 11
COPY M F
@END

COPY M T
NOTE JUMPIF MORE_NUMBERS
FJMP WRITENUMBER

NOTE ELSE HANGUP
COPY -1 #DIAL

NOTE SEEK NEXT NUMBER
SEEK -9999
MULI X 11 T
SEEK T

MARK REDIALER
TEST EOF
FJMP DIALER

NOTE DONE
WIPE
HALT


MARK PARTNER
NOTE CANARY SYN
LINK 800
NOTE SEND SYN ACK
COPY 1 M

GRAB 200
MARK READNUMBER
NOTE SKIP NAME
SEEK 1
@REP 11
COPY F M
@END

COPY T M
FJMP READNUMBER
