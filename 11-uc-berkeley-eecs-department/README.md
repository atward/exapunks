```
GRAB 300
COPY F X

MARK ROOMLOOP
LINK 800
HOST T
TEST T = X

NOTE X = METADATA

NOTE FIND METADATA
GRAB 200

MARK METADATAFINDER
SEEK -3
TEST F = X
SEEK -1
FJMP METADATAFINDER

NOTE TAG
SEEK 1
NOTE SEEK POS
COPY F X
NOTE NUM VALUES
COPY F M

SEEK -9999
SEEK X

NOTE SEND NUM VALUES
COPY M X

MARK READDATA
COPY F M
SUBI X 1 X
TEST X > 0
TJMP READDATA
```
