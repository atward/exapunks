```
GRAB 300
LINK 800

MARK LOOP
COPY F X
REPL FETCHER
TEST EOF
FJMP LOOP

NOTE ACTIVITY+1 SIZE-1
NOTE JUMP BACKOUT

NOTE ACTIVITY-1 SIZE+1
WIPE
HALT

MARK FETCHER
MODE
SWIZ X 3 T
MARK FINDROOM
LINK 800
SUBI T 1 T
TJMP FINDROOM

NOTE READER: CALC FILEID
REPL WRITER
MODI X 100 X
ADDI X 200 X
GRAB X

NOTE GET PEER FILENAME
COPY M X

MARK READFILE
COPY F M
TEST EOF
FJMP READFILE

KILL
DROP
GRAB X

MARK BACKOUT
LINK -1
JUMP BACKOUT


MARK WRITER
MAKE
FILE M
MARK WRITE
COPY M F
JUMP WRITE
```
